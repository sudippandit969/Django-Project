About the Smart Recipe Book Project

The Smart Recipe Book is a Django-based web application designed to simplify cooking and meal planning. Users can add, browse, and manage their personal collection of recipes with ease. The application includes features like recipe creation, photo uploads, and categorized browsing, all wrapped in a modern UI built with Bootstrap 5.
